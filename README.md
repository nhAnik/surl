# Backend API server for a URL shortener

## Setup

1. Clone:
```bash
git clone https://github.com/nhAnik/surl.git
cd surl
```
2. Build and run (`docker compose` should be installed)
```bash
make build
make run
```
3. Migrate the database
```bash
make migrate
```
4. Access the api server at http://localhost:8090.
